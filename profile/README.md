## 🤟 C'est quoi SignReader ?

**SignReader** est un projet open source qui permet aux personnes utilisant la **Langue des Signes Française (LSF)** de se faire comprendre partout dans le monde.

Grâce à la vision par ordinateur et à l'intelligence artificielle, SignReader **détecte les signes en temps réel via la caméra** et les **traduit instantanément en texte** — pour que n'importe qui puisse comprendre, sans connaître la LSF.

> *"Notre objectif : briser les barrières de communication entre la communauté sourde et le reste du monde."*

---

## ✨ Comment ça marche ?

```
📷 Caméra   →   🧠 Modèle IA LSF   →   💬 Texte instantané   →   🌍 Compris partout
```

| Étape | Description |
|-------|-------------|
| 📷 **Capture** | La caméra capte les gestes des mains et du visage en temps réel |
| 🧠 **Reconnaissance** | Notre modèle IA identifie les signes LSF parmi des milliers de patterns |
| 💬 **Traduction** | Le texte correspondant s'affiche instantanément pour l'interlocuteur |
| 🌍 **Communication** | La conversation devient fluide, naturelle et universelle |
