# SignReader 🤟

**Traduire la Langue des Signes Française en texte, en temps réel.**

Chaque main a une voix à faire entendre.
SignReader permet aux personnes utilisant la **Langue des Signes Française (LSF)** de se faire comprendre partout dans le monde en détectant les signes via caméra et en les retranscrivant instantanément en texte.

---

## Comment ça marche

```
📷 Caméra  →  🧠 Modèle IA  →  💬 Texte
```

La caméra capte les gestes en temps réel.
Notre modèle reconnaît les signes LSF et affiche le texte correspondant instantanément, sans latence, sans matériel spécifique.
